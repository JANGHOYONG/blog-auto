# 유튜브 영상 품질 리뷰

`scripts/shorts-generator.js`와 `scripts/longform-generator.js`를 분석해서 아래 항목을 검토하세요.

## 체크 항목

### 1. TTS / 성우
- speakingRate 적절한가 (0.85~0.95 권장)
- 내레이션 길이가 슬라이드/챕터 시간에 맞는가
- 텍스트에 특수문자·영어가 섞여 발음이 어색해질 가능성은 없는가

### 2. 영상 구성
- 슬라이드/챕터 전환 fade 타이밍이 자연스러운가
- 오버레이 자막 글자 수가 화면에 맞는가 (1줄 20자 이내 권장)
- 이미지 검색어(imageQuery)가 건강 주제에 적합한가

### 3. FFmpeg 파이프라인
- complexFilter 구조에 오류 가능성은 없는가
- 오디오/비디오 싱크 문제 발생 지점은 없는가

### 4. GPT 프롬프트
- 스크립트 프롬프트가 시니어 타겟에 맞는 톤인가
- 7챕터 구조(훅→위험→자가진단→해결→반전→실천→마무리)가 잘 지켜지는가

## 출력 형식
각 항목별로 **현재 상태 / 문제점 / 개선 방법**을 구체적으로 제시하세요.
