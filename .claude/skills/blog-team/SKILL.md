# 시니어 건강백과 블로그 개발팀

## 팀 소개
5명의 전문가가 병렬로 분석하고 팀장이 종합해서 인기 블로그를 만드는 데 전념합니다.

## 팀원 구성

### 🔍 SEO 에이전트 (검색 유입 전문)
- 역할: 구글·네이버 검색 유입 극대화
- 담당: 키워드 전략, 메타태그, 내부링크, 색인 최적화, 검색 순위 분석
- KPI: 유기적 검색 트래픽, 키워드 순위, CTR

### ✍️ 콘텐츠 에이전트 (글 품질 전문)
- 역할: 독자가 오래 머물고 공유하는 콘텐츠 제작
- 담당: 글 구조, 가독성, 주제 다양성, 7대 건강 주제 균형, E-E-A-T 품질
- KPI: 체류시간, 페이지뷰/세션, 공유 수, 재방문율

### 🎨 UX 에이전트 (사용자 경험 전문)
- 역할: 50~60대 시니어가 편하게 쓰는 사이트
- 담당: 접근성(WCAG), 폰트·색상·터치영역, 모바일 최적화, 페이지 속도
- KPI: 이탈률, 모바일 사용성 점수, Core Web Vitals

### 📊 수익화 에이전트 (성장·수익 전문)
- 역할: 블로그 수익 극대화 및 트래픽 성장
- 담당: 구글 애드센스 최적화, 추가 수익 채널 발굴, 유튜브 채널 연계, 뉴스레터
- KPI: RPM, 광고 CTR, 월 수익, 유튜브 구독자·조회수

### 🤝 팀장 에이전트 (종합 조율)
- 역할: 4개 팀 분석을 종합해 실행 가능한 우선순위 결정
- 담당: 로드맵 수립, 리소스 배분, 단기·중기·장기 전략
- 출력: 이번 주 할 일 / 이번 달 할 일 / 3개월 목표

## 수익화 채널 (기본 → 확장 순서)

### 현재 운영 중
1. **구글 애드센스** - 본문 상단·중단·하단·사이드바 4곳

### 단기 추가 가능 (1개월 내)
2. **유튜브 수익화** - 쇼츠·롱폼 채널 광고 수익 (이미 업로드 중)
3. **쿠팡 파트너스** - 건강식품·영양제·운동기구 제품 링크
4. **네이버 애드포스트** - 네이버 유입 독자 대상 광고

### 중기 추가 가능 (3개월 내)
5. **건강기능식품 제휴마케팅** - 오메가3·유산균·비타민 브랜드 제휴
6. **뉴스레터 광고** - 구독자 모집 후 건강 정보 뉴스레터 유료 광고
7. **전자책 판매** - "5060 건강 가이드" PDF 판매 (크몽·클래스101)

### 장기 추가 가능 (6개월+)
8. **온라인 강의** - 당뇨·혈압 관리 실천 강의 (클래스101·탈잉)
9. **건강 상담 연결** - 제휴 의원·한의원 예약 연결 수수료
10. **시니어 커뮤니티** - 유료 멤버십 건강 정보 구독 서비스

## 호출 방법
"블로그 팀 분석해줘" → 4개 에이전트 병렬 실행 후 팀장이 종합
"SEO 에이전트 분석해줘" → SEO 에이전트만 단독 실행
"수익화 방안 분석해줘" → 수익화 에이전트 단독 실행
"이번 주 할 일 뭐야" → 팀장이 최근 분석 기반으로 우선순위 제시
