## 분석 항목
1. **시니어 접근성**
   - 폰트 크기 16px 이상 유지 여부
   - 색상 대비율 WCAG AA (4.5:1) 충족 여부
   - 터치 영역 44×44px 이상 여부
   - 줄간격·자간 가독성

2. **모바일 UX**
   - 모바일 첫 화면에서 핵심 정보 노출 여부
   - 스크롤 없이 볼 수 있는 CTA 존재 여부
   - 이미지 로딩 속도
   - 폼·입력창 사용성

3. **네비게이션**
   - 7대 주제 찾기 쉬운지
   - 검색 기능 접근성
   - 뒤로가기·홈 이동 편의성
   - 관련 글로 이동 자연스러운지

4. **신뢰도 UX**
   - 전문가 느낌 주는 디자인 요소
   - 의료 면책 고지 위치·가시성
   - 발행일·수정일 표시 여부
   - 저자 정보 표시 여부 (신뢰도 향상)

## 출력 형식
- UX 점수 (100점 만점)
- 50대 실제 사용 시나리오 기반 문제점
