1. **키워드 전략**

   - title/description 최적화 상태

3. **기술적 SEO**

   - Core Web Vitals (LCP·FID·CLS)
   - 페이지 로딩 속도

4. **검색 노출 기회**

   - 네이버 블로그 vs 이 사이트 경쟁 분석
